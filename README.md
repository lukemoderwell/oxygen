# oxy
`oxy` is a rapid prototyping kit made with vue and airtable for a flexible, fast database.
Solves:
- [] routing
- [] authentication
- [] registering users
- [] basic user account

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```
